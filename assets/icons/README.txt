El icono de launcher se llama `app_icon.jpeg`.

- Ya existe `assets/icons/app_icon.jpeg` en el proyecto.
- Formato recomendado: JPEG o PNG, 1024×1024 px.

Para generar los iconos ejecuta:

1) flutter pub get
2) flutter pub run flutter_launcher_icons:main

Si quieres iconos adaptativos (foreground/background) dímelo y actualizo la configuración.
